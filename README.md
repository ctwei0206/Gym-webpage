# 🏋️ Gym WebPage

A simple interactive fitness web application that helps users manage workouts, track exercise duration, and stay consistent with their fitness routine. The project includes a task-based workout list, countdown timer system, information about gym, facilities, and gym product.

---

## 🚀 Features

- 📝 Add and manage workout tasks
- ⏱️ Built-in countdown timer per workout
- ▶️ Pause, resume, and stop workout sessions
- 📘 More information about gym
- 🏋️ Gym Facilities
- 🛒 Gym Product


---

## 🧠 Tech Stack

- HTML5
- CSS3 
- JavaScript
